# 大耳朵TV风格蒸馏 v0.1

> 状态：`preliminary / Grace(OpenAI)`。本版基于 B 站公开频道页、投稿页截图观察、公开视频 API 元数据与可见标题；尚未做 Claude 交叉验证，也尚未做批量语音转写。

## 1. 频道定位

- 账号：大耳朵TV
- 简介：坐标韩国，主要分享数码产品评测及生活 VLOG。
- 可见数据：粉丝约 111.6 万；投稿视频 990；图文 95；合集和系列 21。
- 核心内容：苹果生态、iOS/iPhone/MacBook/AirPods、AI/Siri、本地 AI、数码配件、购买建议、韩国生活 VLOG、Q&A。

## 2. 第一层风格结论

大耳朵TV不是极简科技频道，而是 **“真人朋友式数码评测 + 强结论标题 + 高信息密度封面 + 生活 VLOG 亲近感”**。

可复用关键词：

> 苹果生态、韩国生活、真实体验、购买建议、强结论、数字化清单、标题长尾关键词、真人出镜、产品近景、高饱和大字、黄/红/绿强调色、箭头圈注、VLOG口吻、轻松但有判断。

## 3. 选题结构

### 高频选题类型

1. **新品/系统更新速评**
   - iOS 新功能、Siri AI、WWDC、iPhone 新机传闻/汇总。
   - 句式：`X 大看点`、`为什么一定要升级`、`严重被低估`。

2. **真实体验挑战**
   - `真实体验48小时`、`连续7天只用...`、`连续三天只使用...`、`挑战24小时...`。
   - 核心不是参数，而是“我真的用了，然后给你结论”。

3. **购买建议 / 避坑**
   - `买前必看`、`不能买 / 放心买`、`为什么最终选择...`、`最大缺点`。
   - 目标受众是要做消费决策的人。

4. **数码好物/EDC**
   - 背包、手机壳、耳机、相机、掌机、Switch、MacBook 配件。
   - 常带 sponsor，但标题仍包装成“值得/不值得”。

5. **纯 VLOG / 韩国生活 / Q&A**
   - 食物、旅行、家庭、人生经验、机场 BB、足球等。
   - 生活内容也用冲突或挑战包装。

## 4. 标题公式

### 主公式

```text
[强结论/强情绪] + [真实体验/时间/数量] + [产品/场景] + [结果/看点] | [关键词串] | 大耳朵TV
```

### 高频标题构件

- 强结论：`太强！`、`严重被低估！`、`终于值得购买！`、`结果令人意外！`、`最大缺点！`
- 时间体验：`48小时`、`连续7天`、`用了2年`、`连续三天`、`24小时`
- 数字清单：`5大看点`、`10大新功能`、`50+主要变化`、`4大优点 / 唯一缺点`
- 决策语：`买前必看`、`为什么一定要升级`、`不能买 / 放心买`
- 搜索词：产品名、系统版本、品牌名、场景词，用竖线分隔。

### 仿写模板

```text
太强！真实体验48小时[产品/功能]，我发现它真的不是噱头！| [关键词1] | [关键词2]
```

```text
【买前必看】用了[X天/年]，我为什么最终选择[型号A]，而不是[型号B]？｜优缺点总结｜[关键词]
```

```text
连续7天只用[旧/冷门产品]，结果我不想换回[主流产品]了！| 真实体验
```

```text
严重被低估！最近[X]年最值得升级的[系统/产品]！| [N]+新功能
```

## 5. 口吻与叙事

- 开场常用亲近招呼：`嗨喽，大家好，今天分享...`
- 立场清楚：先告诉观众“今天我体验/购买/试用了什么”，再给期待。
- 讲评不是纯参数表，而是“生活使用场景 + 主观结论 + 关键参数佐证”。
- VLOG 感强：把评测对象放进真实生活，例如通勤、旅行、吃饭、家庭、韩国日常。
- 情绪尺度：夸张但不低俗；敢用“太强/被低估/不要买/值得买”，但正文需要解释原因。
- 结尾固定动作：感谢收看、喜欢请三连。

## 6. 视频结构模板

### A. 数码评测型 8-14 分钟

1. **0:00-0:20 冲突/结论钩子**：这东西为什么值得/不值得？我真实用了多久。
2. **0:20-1:00 背景交代**：价格、版本、品牌、为什么关注。
3. **1:00-6:00 分点体验**：3-5 个体验点，每点都配真实场景。
4. **6:00-9:00 对比/缺点**：和旧款、竞品、预期对比。
5. **9:00-11:30 购买建议**：谁该买、谁别买。
6. **结尾**：一句总结 + 三连。

### B. 系统/功能更新型

1. 版本/发布时间/适用设备。
2. `N 大变化`清单化。
3. 每个功能用“它解决了什么痛点”解释。
4. 最后给升级建议：稳不稳定、谁适合现在升。

### C. 纯 VLOG 型

1. 一个生活挑战或问题：`连续X天...`、`为什么我...`。
2. 场景连续推进：出门/购买/使用/吃饭/聊天。
3. 插入 Q&A 或人生经验。
4. 结尾回到一个轻结论：好不好、值不值、有没有改变看法。

## 7. 封面风格

基于投稿页截图观察：

- 16:9 横图，高信息密度。
- 真人主播高频出镜：手持产品、指向产品、惊讶/认真/皱眉表情。
- 产品近景：iPhone、MacBook、耳机、系统界面、配件，占画面核心区域。
- 大号中文粗体字，常带描边/阴影。
- 高频颜色：黄色、红色、绿色、蓝紫科技感背景、白字描边、黑色/暗色压底。
- 图形符号：红箭头、圈注、对比数字、红叉绿勾、版本号/价格。
- 封面字不追求完整语法，追求 4-10 字的强卖点：`最大看点`、`不要买`、`值得买`、`提升100倍`。

### 封面模板

```text
左侧：黄色/白色大字结论
右侧：主播 + 产品特写
中间：红箭头/圈注指向关键细节
角落：型号、价格、版本号或“N大变化”
整体：高饱和、高对比、B站科技区点击感
```
