## 一句话判断

## 笔记提到的资源

### 需要申请 / Startup Program

| 项目 | 笔记声称额度 | 适合用途 | Mia 判断 |
|---|---:|---|---|
| Claude for Startups / Anthropic | 25K API credits；有 VC/partner 推荐可到 100K+；12 个月 | Claude API 原型、Agent 后端、批处理 | 很值得优先查官网，尤其适合 AI-first 项目 |
| OpenAI for Startups | 视项目阶段而定，通常有门槛 / partner 牵线 | GPT API、语音、多模态、评测 | 有用，但门槛和入口可能变化快 |
| Google for Startups Cloud | 最高 350K；Vertex AI 可跑 Gemini/Claude | 云资源、Gemini、托管服务 | 适合真正要跑云服务/训练/部署的项目 |
| AWS Activate | 最高 300K；Bedrock 可用 Claude/Mistral/Llama | Bedrock、多模型评测、后端部署 | 对工程团队有用，需看资质和 credits 可用范围 |
| Microsoft Founders Hub | 最高 50K Azure；Azure OpenAI | GPT 系列、Azure 云服务 | 申请路径相对清晰，适合早期公司 |

### 不用申请 / 免费层

| 项目 | 笔记描述 | 适合用途 | Mia 判断 |
|---|---|---|---|
| Groq | free tier、限速、不一定要信用卡 | 超快推理、demo、原型 | 适合作为备用推理通道 |
| Cerebras | free tier、限速 | 快速推理实验 | 可试，但生态/模型选择需确认 |
| Google Gemini | free tier | 低成本原型、多模态 | 适合日常实验 |
| OpenRouter | free tier / 免费模型 | 多模型路由与试用 | 对 Agent 工具链很实用，但注意模型稳定性 |
| HuggingFace | free tier | 模型/数据集/Space demo | 适合开源模型与 demo 托管 |

## 对用户项目的价值

1. **如果做 Agent / AI 产品原型**：优先关注 Claude、OpenAI、Gemini、OpenRouter、Groq。  
2. **如果要上线真实服务**：云厂商 credits 比单纯 API credits 更重要，因为还会用到数据库、队列、对象存储、日志、部署。  
3. **如果只是个人实验**：先用免费层和低门槛入口，不急着申请复杂 startup program。  
4. **如果未来要公司化**：这些额度可以整理成一张申请表，按「公司主体、官网、pitch deck、VC/孵化器推荐、云账号」逐项准备。

## 风险 / 注意

- 小红书内容没有逐项官方核验；额度可能随地区、时间、申请入口变化。  
- “最高额度”通常不是默认额度，往往需要公司资质、融资状态、合作伙伴推荐或审核。  
- credits 可能限制用途：只能云资源、只能 API、不能转售、不能用于某些模型。  
- 对 Agent 项目不要只看免费额度，还要看：速率限制、上下文长度、稳定性、数据隐私、商用条款、账单上限。
