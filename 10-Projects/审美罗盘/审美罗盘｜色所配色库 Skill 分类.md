# 色所配色库｜Skill 开发分类

用途：这是给 **Aesthetic Compass / 审美罗盘** 调用的配色分类，不是给用户日常浏览 Obsidian 用的笔记。

## 总规则

- 这些素材只作为 `color_only` 配色证据。
- 不继承原公众号海报的布局、字体、构图。
- 每组配色进入产品前必须转译为 `surface / brand / accent / ink / state`。
- 高饱和、强撞色、品牌案例类不能直接全局铺色，要先做对比度和场景约束。

## Skill 调用策略

| 用户需求 | 优先调用族群 | 使用方式 |
|---|---|---|
| “高级、干净、耐看、长期使用” | `muted-soft` / `analogous-system` / `premium-brand` | 做基础主题、色阶、surface 与 brand token |
| “AI、科技、未来感” | `tech-cyber` / `gradient-atmosphere` | 深色主题、hero 氛围、动效背景 |
| “年轻、活泼、传播感” | `dopamine-playful` / `classic-contrast` | 局部 accent、营销页、活动模块 |

| “贵、高端、会员感” | `luxury-dark` | 深色 surface + 金属/宝石 accent |
| “参考某品牌的色彩关系” | `brand-reference` | 只学习色彩结构，不复制品牌资产 |
