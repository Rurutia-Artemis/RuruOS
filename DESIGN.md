# DESIGN.md · 糖霜设计宪法

RuruOS 的视觉风格叫**糖霜（Tangshuang）**：糖果色玻璃拟态手账。奶油底、糖果六色各司其职、圆体大字、贴纸与虚线、樱花飘落。可爱是包装纸，纪律在里面。

这份文档写给想改界面的人类；用 AI 改样式时不用背这些——库里自带 `.claude/skills/tangshuang/` 风格规范，Claude Code 会自动遵守。**两份规范冲突时以 skill 为准。**

## 为什么锁这么死

这套风格前后经过几十轮真机迭代才定稿，每条规矩背后都是一个踩过的坑。改样式最快的毁法不是改丑一个按钮，而是「各改一点」造成的漂移——三个月后没有一个页面像同一个系统。所以：**个性化走合法旋钮（六主题、色相滑杆、CSS snippet），不改母版。**

## 糖果六色（hex 禁改，角色禁挪用）

| 色 | Hex | 角色 |
|---|---|---|
| 莓果 berry | `#ff8fab` | 主强调、警示/删除 |
| 蜜桃 peach | `#ffab76` | 选中态、标题强调 |
| 奶油 butter | `#ffd66b` | 荧光笔、便签、数字 |
| 薄荷 mint | `#6fd8b4` | 成功、确认 |
| 天空 sky | `#7db8f7` | 链接、信息、focus 描边 |
| 香芋 taro | `#b9a5f5` | 次级、引用 |

主题底六套：cream / matcha / soda / peachtea / berrynight / grapenight，默认 cream。

## 字体与数字

- 标题、大数字：圆体栈 `'Yuanti SC','Hiragino Maru Gothic ProN','PingFang SC'`，weight 800
- 正文：系统栈，行高 1.7 起
- 对齐数字一律 `font-variant-numeric: tabular-nums`

## 组件铁律（精选）

1. **胶囊按钮**：12px 圆角矩形；未选中 = 功能色 16% 混糖霜底 + 功能色细边（**严禁裸背景黑药丸**）；选中 = 糖果实底。一组 tab 一 tab 一色。
2. **果冻主钮**：顶部提亮渐变 + 双层阴影，hover 上浮 2px、按下压缩 0.97。
3. **卡片**：圆角 26px、半透明底 + backdrop-blur；内部分隔线一律 2px 虚线。
4. **图标**：必须手绘场景化 SVG（viewBox 24、深描边、粉彩填充、圆头线帽）。emoji 和 Unicode 字符（⇅ ✿ ◐）当图标一律不行。
5. **粒子层与一切装饰动画层**：容器必须自带 `overflow: hidden`（否则粒子飘出边界把滚动条挤出来，整页宽度呼吸式抖动）；`prefers-reduced-motion` 时整层关闭，这条不许省。
6. **动效曲线**只用两条：弹性 `cubic-bezier(0.34,1.56,0.64,1)`，丝滑 `cubic-bezier(0.22,1,0.36,1)`。
7. **调节项多于两个**不许裸摆在栏上——收进一颗按钮的弹出面板。
8. **布局稳定三件套**：内滚动容器 `scrollbar-gutter: stable`；轮换文字按最大行数预留 `min-height`；body 渐变背景禁用 `fixed` attachment。
9. **签名元素**（日签卡、拍立得墙这个量级的自定义造物）每页限一个；胆量花在一处，不许每个区域都上动效。
10. **空档治理**：容器高度贴内容走，不许用 `space-between` 把空档摊到中间；层间距超过 24px 就是没治好。

## 禁止清单

- 未选中胶囊无底色（黑药丸）
- 靛蓝/紫渐变大背景、居中白底蓝按钮的「模板脸」
- emoji / Unicode 字符当图标
- 每幕都上动效
- 在单页/单处就地改母版 token（要动 token 只改母版，再整体同步）

## 改动后的自检

- [ ] 六套主题逐一切过，深色两套下所有按钮边界清晰
- [ ] 粒子层在 reduced-motion 下关闭

- [ ] 数字全部 tabular-nums
- [ ] 签名元素 ≤ 1 个
