# 自定义指南

RuruOS 的个性化分两层：**界面里点点就能改的**，和**改一行配置的**。改样式之前请先读根目录 `DESIGN.md`——这套系统好看是因为纪律，不是因为随意。

## 界面内就能改（无需碰代码）

| 想改什么 | 在哪改 |
|---|---|
| 主题（四浅二深六套） | 顶栏「主题」方章 → 弹出面板选择，还有色相滑杆微调 |
| 桌面动效（樱花/细雨/落雪/萤火） | 顶栏「动效」方章 → 速度/数量/大小/方向/饱和度/可见度六项可调 |
| 你的名字（顶栏名牌） | 直接点名字，行内编辑 |
| 天气城市 | 点天气卡的**城市胶囊**（别针小药丸）→ 搜城市或自动定位；首次启动会按网络 IP 自动定一次 |
| 系统级全屏（只留中枢） | 顶栏「全屏」方章，再点一次退出 |
| 界面整体字号 | 顶栏「A− % A＋」胶囊 |
| 阅读器字号/版宽/字体/标题色/正文明度 | 阅读器顶栏「Aa 外观」按钮 |

以上全部自动记忆（localStorage），换设备不跟随。

## 改一行配置的（`.obsidian/plugins/obos-home/main.js`）

打开 main.js，顶部有一段**「公开版配置区」**注释块，能改的都在那里和下面两处。改完 `Cmd/Ctrl + P` → 「重新加载」生效。

1. **项目卡目录**：`PROJECTS_DIR = ''` → 填你的代码目录绝对路径（如 `'/Users/you/Code'`），中枢项目卡就会只读扫描它
2. **AI 引擎**：想强制用某家就填 `AI_ENGINE_FORCE`（如 `'codex'`）；工具装在非常规路径就往 `AI_ENGINES` 对应项的 `bins` 数组加路径；接列表外的新工具就照格式加一项（详见 `docs/接入AI指南.md`）

（天气城市已经不在这里了——界面里点城市胶囊就能换，见上表。）

## 换句库

顶栏金句轮换自 `90-System/句子精选.md`，格式一行一句：`句子｜出处`。整个文件换成你自己喜欢的句子即可，5 分钟自动轮换，顶栏 ↻ 手动换。

## 换徽记

顶栏左上角的果冻手账本 logo 是 `.obsidian/plugins/obos-home/crest.png`，换成你自己的透明底 PNG（建议 512px）即可。

## 想动样式？先读这个

- 全部设计 token（颜色/圆角/阴影）集中在 `styles.css` 顶部 `.obos-root` 的 CSS 变量区——**这一区是母版，不要就地改**
- 自定义样式的正确姿势：Obsidian 设置 → 外观 → CSS 代码片段，新建你自己的 snippet 去覆盖变量，永远不碰 `styles.css` 本体（这样更新插件不会丢你的定制）
- 颜色角色、组件规矩、禁止清单见根目录 **`DESIGN.md`**；用 AI 改样式的话，AI 会自动读 `.claude/skills/tangshuang/`（糖霜风格规范）替你守规矩
