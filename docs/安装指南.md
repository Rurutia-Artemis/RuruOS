# 安装指南

> **最省事的方式是让 AI 替你装**：把仓库地址发给你的 AI，让它按根目录《AI搭建指南.md》执行。本文是人类手动路线。

从零开始，到看见完整的 RuruOS 中枢。**不需要任何 AI 也能用**——AI 只是锦上添花，先装好基本盘。

## 第一步：拿到这个库

两种方式任选：

- **Git**（推荐，以后能跟着更新）：`git clone https://github.com/Rurutia-Artemis/RuruOS.git`
- **ZIP**：GitHub 页面点绿色 `Code` 按钮 → `Download ZIP` → 解压到你喜欢的位置

## 第二步：用 Obsidian 打开

1. 下载安装 [Obsidian](https://obsidian.md)（免费）
2. 启动 → 「打开另一个仓库」→ 「打开本地仓库文件夹」→ 选中刚才的 `RuruOS` 文件夹
3. 首次打开会问是否信任此仓库并启用社区插件——选**信任**（中枢界面就是靠 `obos-home` 这个随库自带的插件驱动的；代码就在 `.obsidian/plugins/obos-home/`，全部明文可审查）
4. 如果界面没有自动进入中枢：`Cmd/Ctrl + P` 打开命令面板 → 输入「中枢」→ 执行 **打开 RuruOS 中枢**

这时你应该看到：糖果色顶栏（名牌、金句、时钟、天气、小黄历）、待办卡、日历卡、账目卡、文章卡、追剧墙。库里预置了少量示例数据（三本虚构账本、示例流水等），看完效果随时可删。

## 第三步：选一条路线

### 路线 A：不用 AI（零依赖）

装完即用。你手上有的功能：

| 功能 | 依赖 |
|---|---|
| 中枢全部界面、六套主题、粒子动效、字号调节 | 无 |
| 待办 / 日历（界面内直接增删改） | 无 |
| 账本 / 流水（账目卡汇总、账本详情页：占比条/对账单/时间线） | 无（录入见下方说明） |
| 搜剧建档（TVmaze + 豆瓣联想、九宫格选海报、分季） | 免费公开 API，无需注册 |
| 天气卡（Open-Meteo） | 免费公开 API，无需注册 |
| 文章阅读器（进度记忆、备注便利贴、Aa 外观面板） | 无 |
| 苹果 提醒事项/日历 同步（macOS） | 本机 python3，首次点「同步」按钮时系统会请求授权 |

没有 AI 你缺的：文章自动抓取入库、AI 浓缩、AI 清杂、剧集资料自动补全、中文剧名转译兜底，以及**记账录入**——RuruOS 刻意不做记账表单，账目的正门是把 Agent 当私人管家：你说一句「昨天花了多少」，它按 `docs/记账指南.md` 写好文件，中枢自动渲染成漂亮的账本页。替代方法（纯手动）：文章自己粘贴成 markdown 按模板填 frontmatter（格式见根目录 `AGENTS.md`）；账目照《记账指南》的模板手写 md 文件，效果完全一样，只是没人替你动手；剧集资料在详情页点「联网搜封面」「搜评分」旁的手动按钮，或干脆手填。

### 路线 B：配一个本机 agent CLI（完整体验，推荐）

Claude Code、Codex、Gemini CLI、Qwen Code（通义）、iFlow（心流）、OpenCode——装任意一个并登录，插件里所有 AI 按钮自动探测激活。同时你可以直接在库目录里跑它，让它按库规范帮你整理内容。详见 `docs/接入AI指南.md`。

### 路线 C：配其他智能体（Codex、Grok、Hermes、ChatGPT 等）

插件内的 AI 按钮只认 claude CLI，但「往库里写内容」这件事任何智能体都能干——把根目录的 **`给智能体的说明书.md`** 全文喂给它即可。详见 `docs/接入AI指南.md`。

## 常见问题

**Q：装出来是紧凑深色的旧样子，或干脆是 Obsidian 默认界面，没有糖果色中枢？**
按命中率排查三件事：

1. **`.obsidian/plugins/` 里有没有第二个 obos-home**（比如 `obos-home.bak-xxxx/`、`obos-home 副本/`）。Obsidian 按 manifest 里的 `id` 识别插件，两个文件夹同 id 时哪个被加载不由你定——实战翻过车：一个旧备份长期遮蔽真身，界面一直跑旧代码，重启也没用。把备份移出 `.obsidian/plugins/`（放到库外任何地方）再重启 Obsidian 即可。终端一句话自查：`ls <库路径>/.obsidian/plugins/`，看到两个 `obos` 开头的文件夹就是中招了。
2. **`.obsidian` 隐藏文件夹没拷全**。ZIP 解压后用访达/资源管理器手动拖文件时，隐藏的 `.obsidian` 容易被漏掉——Obsidian 会自动生成一套默认配置（默认深色、零插件）。确认库文件夹里有 `.obsidian/plugins/obos-home/`（macOS 访达按 `Cmd+Shift+.` 显示隐藏文件），没有就重新 `git clone`，并直接用 Obsidian 打开克隆出来的文件夹。
3. **受限模式**。设置 → 第三方插件，如果显示「受限模式」，关闭它并启用 obos-home。

**Q：点 AI 按钮弹「未检测到本机 AI 命令行工具」？**
正常——你还没装任何 agent CLI，AI 按钮不可用，其余功能不受影响。想用就走路线 B（六款工具任选其一）。

**Q：项目卡是空的？**
项目卡默认关闭（它扫描的是你本机的代码目录）。想开启：编辑 `.obsidian/plugins/obos-home/main.js` 顶部「公开版配置区」的 `PROJECTS_DIR`，填你的代码目录绝对路径，然后 `Cmd/Ctrl + P` → 「重新加载」。

**Q：天气显示的城市不对？**
首次启动会按网络 IP 自动定位一次（挂 VPN/代理时会定到出口城市，属正常）。点天气卡上的**城市胶囊**（别针小药丸）→ 搜城市（中文/拼音/英文）或自动定位，选一次永久记住，全程不用碰代码。

**Q：多设备同步？**
用 Obsidian Sync 的话，记得在 Sync 设置里打开「同步已安装的社区插件」，否则中枢插件不会跟着走。用 iCloud/git 同步整个文件夹也可以。

**Q：改了插件代码怎么生效？**
`Cmd/Ctrl + P` → 搜「重新加载」（Reload app without saving）执行即可，无需重启。

**Q：手机上能用吗？**
数据层是纯 markdown，手机 Obsidian 能读写所有笔记；但中枢插件是为桌面大屏设计的，移动端未适配。
