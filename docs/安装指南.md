# 安装指南

从零开始，到看见完整的 RuruOS 中枢。**不需要任何 AI 也能用**——AI 只是锦上添花，先装好基本盘。

## 第一步：拿到这个库

两种方式任选：

- **Git**（推荐，以后能跟着更新）：`git clone https://github.com/Rurutia-Artemis/RuruOS.git`
- **ZIP**：GitHub 页面点绿色 `Code` 按钮 → `Download ZIP` → 解压到你喜欢的位置

## 第二步：用 Obsidian 打开

1. 下载安装 [Obsidian](https://obsidian.md)（免费）
2. 启动 → 「打开另一个仓库」→ 「打开本地仓库文件夹」→ 选中刚才的 `RuruOS` 文件夹
3. 首次打开会问是否信任此仓库并启用社区插件——选**信任**（中枢界面就是靠 `obos-home` 这个随库自带的插件驱动的；代码就在 `.obsidian/plugins/obos-home/`，全部明文可审查）
4. 如果界面没有自动进入中枢：`Cmd/Ctrl + P` 打开命令面板 → 输入「中枢」→ 执行 **打开 RuruOS 中枢**

这时你应该看到：糖果色顶栏（名牌、金句、时钟、天气、小黄历）、待办卡、日历卡、账目卡、文章卡、追剧墙。库里预置了少量示例数据（三本虚构账本、示例流水等），看完效果随时可删。

## 第三步：选一条路线

### 路线 A：不用 AI（零依赖）

装完即用。你手上有的功能：

| 功能 | 依赖 |
|---|---|
| 中枢全部界面、六套主题、粒子动效、字号调节 | 无 |
| 待办 / 日历 / 账本 / 流水（界面内直接增删改） | 无 |
| 搜剧建档（TVmaze + 豆瓣联想、九宫格选海报、分季） | 免费公开 API，无需注册 |
| 天气卡（Open-Meteo） | 免费公开 API，无需注册 |
| 文章阅读器（进度记忆、备注便利贴、Aa 外观面板） | 无 |
| 苹果 提醒事项/日历 同步（macOS） | 本机 python3，首次点「同步」按钮时系统会请求授权 |

没有 AI 你缺的：文章自动抓取入库、AI 浓缩、AI 清杂、剧集资料自动补全、中文剧名转译兜底。替代方法：文章自己粘贴成 markdown 按模板填 frontmatter（格式见根目录 `AGENTS.md`）；剧集资料在详情页点「联网搜封面」「搜评分」旁的手动按钮，或干脆手填。

### 路线 B：配 Claude Code（完整体验，推荐）

装 [Claude Code](https://claude.com/claude-code) 并登录后，插件里所有 AI 按钮自动激活（会自动探测 `claude` 命令的位置）。同时你可以直接在库目录里跑 `claude`，让它按库规范帮你整理内容。详见 `docs/接入AI指南.md`。

### 路线 C：配其他智能体（Codex、Grok、Hermes、ChatGPT 等）

插件内的 AI 按钮只认 claude CLI，但「往库里写内容」这件事任何智能体都能干——把根目录的 **`给智能体的说明书.md`** 全文喂给它即可。详见 `docs/接入AI指南.md`。

## 常见问题

**Q：点 AI 按钮弹「未检测到 claude CLI」？**
正常——你没装 Claude Code，该按钮不可用，其余功能不受影响。想用就走路线 B。

**Q：项目卡是空的？**
项目卡默认关闭（它扫描的是你本机的代码目录）。想开启：编辑 `.obsidian/plugins/obos-home/main.js` 顶部「公开版配置区」的 `PROJECTS_DIR`，填你的代码目录绝对路径，然后 `Cmd/Ctrl + P` → 「重新加载」。

**Q：天气显示的是东京？**
默认坐标是东京。改法见 `docs/自定义指南.md`（两分钟搞定）。

**Q：多设备同步？**
用 Obsidian Sync 的话，记得在 Sync 设置里打开「同步已安装的社区插件」，否则中枢插件不会跟着走。用 iCloud/git 同步整个文件夹也可以。

**Q：改了插件代码怎么生效？**
`Cmd/Ctrl + P` → 搜「重新加载」（Reload app without saving）执行即可，无需重启。

**Q：手机上能用吗？**
数据层是纯 markdown，手机 Obsidian 能读写所有笔记；但中枢插件是为桌面大屏设计的，移动端未适配。
