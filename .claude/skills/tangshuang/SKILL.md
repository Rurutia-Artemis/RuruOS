# 糖霜（Tangshuang）· 软糖手账网页风格

一句话：糖果色玻璃拟态手账。奶油底、糖果六色各司其职、圆体大字、贴纸与虚线、樱花飘落。可爱是包装纸，纪律在里面：**签名元素每页只许一个，其余全部用标准组件**。

## 强制流程

1. **从母版起步**：Read `assets/ruru-template.html`——这是全 skill **唯一的样式母版**。复制为新页起点，内容写进 `<main>`。`<style>` 分两区，用注释硬线隔开：`╔══ 母版区 ══╗` 里的 tokens / 六主题 / 组件 CSS **逐字禁改、原样保留**；本页专属样式一律写在 `╚══ 母版区结束 ══╝` 之后。要动 tokens 或组件，**只改母版再整段同步到各页**，别在单页里就地改样式——四条历史漂移全是这么来的。
2. **看示例校准**：改动前 Read `assets/example-homepage.html`（个人主页示例，母版区与母版逐字一致），对齐组件用法和排版密度。
3. **方案有分歧就做候选页**（v7.0 定流程）：图标/布局/配色/动效拿不准时，用本骨架做 1:1 真实渲染的 HTML 候选页——编号标注（A/B/C、S1-S4…）、每项一句说明、底部写「怎么选」回复示例——发给用户看效果拍板。文字描述和 ASCII 示意图传达不了质感，用户明确毙过。**发 claude.ai Artifact 预览注意**（v7.1 实锤）：Artifact 托管会把静态 CSS 里的 `color-mix()` 全部剥坏（果冻钮变白板、渐变字隐形，本地浏览器无此问题）——预览版要把整段 CSS 塞进 `<script>` 用 `document.createElement('style') + textContent` 运行时注入，并补 `document.body.dataset.theme = ... || 'cream'` 兜底；正式交付文件不受影响，**不要为此改母版**。
4. **发布前过自检清单**（本文末尾），有一条不过就修。

## 色板（禁改 hex）

糖果六色，角色固定，不许挪用：

| 色 | Hex | 角色 |
|---|---|---|
| 莓果 berry | `#ff8fab` | 主强调、警示/删除 |
| 蜜桃 peach | `#ffab76` | 选中态、标题强调 |
| 奶油 butter | `#ffd66b` | 荧光笔、便签、数字 |
| 薄荷 mint | `#6fd8b4` | 成功、确认 |
| 天空 sky | `#7db8f7` | 链接、信息、focus 描边 |
| 香芋 taro | `#b9a5f5` | 次级、引用 |

主题底六套（cream / matcha / soda / peachtea / berrynight / grapenight），hex 见骨架文件，页面必须带主题切换。默认 cream。

## 字体与数字

- 标题、大数字：`'Yuanti SC','Hiragino Maru Gothic ProN','PingFang SC'`，weight 800
- 正文：`-apple-system,'PingFang SC'`，行高 1.7 起
- 对齐数字一律 `font-variant-numeric: tabular-nums`

## 组件铁律

1. **全彩胶囊 `.pill`**（v6.5 拍板升级）：形状统一 **12px 圆角矩形**，不用药丸形；未选中 = 功能色 16% 混糖霜底 + 功能色 45% 细边 + ink 字（**严禁裸背景黑药丸**）；选中 `.on` = 糖果实底 + `--sticker` 深字 + 果冻双层阴影。功能色用 `--pc` 传，**一组 tab 一 tab 一色**（如剧集五态：待定黄/想看香芋/在追天蓝/看完薄荷/弃剧莓果）。
2. **果冻主钮 `.jelly`**：顶部提亮渐变 + 双层阴影，hover 上浮 2px、按下下压缩 0.97；语义色用 `.jc-berry/.jc-mint/...` 修饰类。
3. **卡片 `.canvas`**：圆角 26px、半透明底 + backdrop-blur、1.5px 边；内部分隔线一律 `2px dashed var(--line)`。
4. **贴纸标签 `.act-tag`**：分幕标题用它，`rotate(-1deg)` 胶带感，`--tag` 传角标色。
5. **便利贴 `.note`**：`--note-bg/--note-bd` 变量派生，主题联动，不许写死黄色。
6. **粒子层**：樱花/细雨/落雪/萤火四种，参数六项可调——`fxState = { mode, speed, count, size, angle, sat, vis }`（angle 方向角 -45°~45° 负值左斜；sat 饱和度、vis 可见度均为 0.3~2 乘子，层级变量拖动零闪烁）；浅色主题下雨/雪/萤火配色自动加深一档；`prefers-reduced-motion` 时整层关闭，这条不许省。**粒子层和一切装饰动画层（光斑泡泡/上浮糖粒）容器必须自带 `overflow: hidden`，别赌宿主会裁剪**——粒子飘出边界会把滚动条挤出来又收回，整页宽度呼吸式抖动（RuruOS v6.9b 实锤 bug，探针查三轮才抓到）。
7. **动效曲线**：弹性 `cubic-bezier(0.34,1.56,0.64,1)`，丝滑位移 `cubic-bezier(0.22,1,0.36,1)`，不用别的。
8. **参数面板 `.pop`**：调节项多于两个不许裸摆在栏上——收进一颗按钮，点开糖果弹层（玻璃卡 + 贴纸标题 + 每行「标签/滑杆/值」+ 底部恢复默认），点外部关闭，`popin` 弹性入场；面板底要近实底（`color-mix(var(--bg) 88%, #fff)`），太透会跟后面内容搅在一起。桌面动效的六项调节就是标准用例。
9. **方章功能钮**（v7.0 质感定稿）：顶栏级主功能按钮用 **65px 圆角方章（18px 圆角）**，果冻渐变一钮一色 + 三层质感：顶部玻璃高光带（`::before` 左右缩进 7px、高 34%、下缘 46% 圆角、白 0.6→0.05 渐变）+ 底部内阴影压出果冻厚度（`inset 0 -5px 10px` 功能色 62% 混 `#7a4630`）+ 糖底硬边（`0 4px 0` 深一档）+ 彩色弥散投影。**图标必须手绘场景化 SVG（40px，占钮面 60%+；viewBox24、深描边、粉彩填充、圆头线帽）**，Unicode 字符（⇅ ✿ ◐）与 emoji 都不行——用户毙过两轮才定稿。定稿参考三颗：同步=大循环环+中心糖点、动效=飘落三樱瓣+风线、主题=调色盘四色点（SVG 全文在骨架示例区，可直接复用）。
10. **空档治理**（v6.5 血泪版）：容器高度贴内容走——不许用 `space-between` 把空档摊到中间，更不许把内容吹大去填过高的容器；两栏高度失衡时优先改**左右结构**（如天气图标和温度并排）消高差。改完必须实测：层间距超过 24px 就是没治好。
11. **等高信息块**：同组统计块 `grid-auto-rows: 1fr` 强制等高，内容少的块用数据行补齐（每块 1-2 条实时数据 + 辅助行），块内版式=大数字居左 3em + 右侧标签/数据列 + 左缘 5px 彩条。
12. **数值语义色**：温度类数值用渐变文字（`background-clip: text`）按数值分档，禁纯色。温度五档：≤5° 天蓝→香芋，6-15° 天蓝→薄荷，16-24° 薄荷→奶油，25-31° 奶油→蜜桃，≥32° 蜜桃→莓果。
13. **渐变名牌字 `.grad-name`**（v7.0 拍板）：人名/品牌词这类情感主角用两行名牌制——上行 muted 小字（时段问候/身份，800 字重带字距），下行大号糖果渐变字：圆体 2.7em + `linear-gradient(118deg, berry, peach 48%, taro)` + `background-clip: text` + berry 30% drop-shadow 光晕。渐变字不配下划线（虚线更不行，会打架）；可点击元素用光晕加深做 hover 态。
14. **布局稳定三件套**（v6.8-6.9 血泪版）：内滚动容器加 `scrollbar-gutter: stable`（滚动条出收不再挤宽度）；轮换文字（金句/口号）按最大行数预留 `min-height` 并在框内上下居中，短句长句切换页面不跳；**body 渐变背景禁用 `fixed` attachment**——Artifact/iframe 里会断层露白，用 `min-height: 100vh` + 普通渐变替代。

## 排版模式（严格照此结构）

```
sticky 控制条（徽记 + 页面名 + 主题切换球）
└ main.stage（max-width 1240，两侧 clamp 留白）
   ├ 幕①：.act-tag 贴纸标题 → 一句引导文 → .canvas 画布卡
   ├ 幕②：同上，幕与幕间距 64px
   └ …（每幕一个主题，不混）
```

- 大数字放 `.stat-num` 规格（圆体 2.6em + tabular-nums + 顶部色相光晕）
- 说明性小字用 `.dev-note` 规格（0.8em、muted 色）
- 签名元素（日签卡/拍立得墙这个量级的自定义造物）**每页限一个**，放最重要的幕

## 动效模式库（v7.1 拍板，2026-07-13）

参考实现全部在 `assets/motion-patterns.html`（1:1 可跑，每段 CSS/JS 带模式名注释，直接拷对应块）。签名元素限一个的规矩不变：主角类模式选一个当主角，氛围类/进场类不占签名额度。

**进场类**
1. **开场编排**（hero 一次性，⚠️ 手感仍在迭代）：进场必须分角色——问候语轻落（0.3s）→ 名牌字从基线破土（clip-path 遮罩 + translateY，0.5s 丝滑曲线）→ 按钮小步上浮（0.32s，60ms 错峰）→ 装饰糖果最后弹簧落座。主体 0.9s 内收完。**弹簧只给糖果类装饰，不给字**——字上弹簧显廉价（用户毙过）。落成后可借「流光扫过」当点睛（单次，见主角类 4）。
2. **滚动飞入 + 贴纸盖章**（幕与项目卡）：卡片 `translateY(44px) rotate(2.5deg) scale(0.94)` → 复位，0.65s 弹性曲线；角标贴纸 `scale(1.7) rotate(-10deg)` 延迟 0.25s「拍」下去；IntersectionObserver 触发，离开视野可复位以便反复观看。

**氛围类**（不占签名额度，可与任意主角叠）
3. **景深糖云**（背景层）：四团糖色 `radial-gradient` 光云（150-290px，`blur(26px)`，功能色 55% 混透明），各自慢漂 15-23s 交错；指针移动时按层视差（factor 0.05-0.2，外层 transition 0.35s 丝滑）。容器必须 `overflow: hidden`（铁律 6）。
4. **爆糖开关**（全站彩蛋，收进动效面板当开关，**默认关**）：鼠标糖尾（花瓣形小粒 0.9s 淡出，节流 ~36ms）+ 点击爆糖（12-14 粒极坐标散开 0.75s）。粒子 `animationend` 即移除。

**主角类**（每页选一个当冲击力主角）
5. **流光巨字**：名牌字放大到 `clamp(4em, 11vw, 7em)`，糖果渐变 `background-size: 220%` 慢流 5s linear；**另叠一层同文字的白高光（透明→白 0.95→透明，105deg），每 3.6s 扫过一次**——没有这道「闪」用户看不出在动（v7.1 实锤）；角落两颗星光 SVG 呼吸。双层用 grid 同格叠放。
6. **拍立得手账墙**：白框 `#fffdf8` + 底部留白写手账字（固定深棕，不随主题），和纸胶带 = 功能色 42% 混半透白、微旋转；三张各自旋转 -7°~6° 叠放；hover 摆正抬起放大（0.22s 丝滑 + 阴影加深 + z-index 提层）。照片位先用糖果色 CSS 场景占位，交付时换真照片/截图。
7. **可戳的巨型果冻**：徽记放大成真果冻——外层弹性跟随指针（`transform 0.3s` 弹性曲线，位移 = 指针偏移 × 0.14），内层按压压扁 `scale(0.86, 0.92)`（0.12s）+ 迸出 1-2 片花瓣，松手弹簧回弹自带颤感；内层再叠 4s 呼吸。外层跟随、内层压扁分两层做，transform 才不打架。
8. **日签卡**：手账「今日一页」——`perspective: 1100px` + `rotateY(180deg)` 翻面 0.56s 丝滑曲线，双面 `backface-visibility: hidden`；正面 = 活日期（JS 算）+ 一句话 + 东京邮戳（虚线圆 `rotate(-12deg)` 小字），背面 = 今日在忙。

**动效手感铁律**（本轮血泪）
- 悬停效果一律 `@media (hover: hover) and (pointer: fine)` 门控，触屏不误触发。
- 按压反馈 ≤160ms、进场主体 ≤0.9s、错峰 30-80ms；纯装饰循环（流光/糖云/呼吸）可以慢。
- 均匀无主次的「所有元素同一套缩放弹入」直接毙——进场必须分角色分曲线。
- **用户毙过，别再提**：果冻积木字（名字拆软糖块，精修版也毙）、玻璃糖罩（展品罩，没必要）。

## 禁止

- 未选中胶囊无底色（黑药丸）
- 靛蓝/紫渐变大背景、居中白底蓝按钮的模板脸
- emoji 或 Unicode 字符（⇅ ✿ ◐ 这类）当图标——一律手绘场景化 SVG（v7.0 收紧：字符符号也被用户毙了）
- 引号用 `::before/::after` 伪元素挂（配 grid/flex 会被拆到独立行）——直接写进文本
- 每幕都上动效（胆量花在一处）
