# 记账首页

RuruOS 的账目分两层，schema 的唯一权威在根目录 `AGENTS.md`：

1. **流水**（`流水/`）：日常收支一笔一笔记，文件名 `YYYY-MM-DD 标题.md`，frontmatter 带 `amount / currency / direction / category`。中枢账目卡自动汇总本月流水。
2. **项目账本**（本目录）：大项目的持续账本，`type: ledger`，frontmatter 的 `total` 与 `currency` 是中枢读取的字段——**正文更新后必须同步更新这两个字段**。欠款类账本 `total` 直接写负数，中枢会红色显示。

账本正文就是普通 markdown 表格，插件会自动变形：

- 带「占比」列的表格 → 分段占比条
- 两列且 ≤6 行 → 统计胶囊
- 其他表格 → 对账单式明细表（斑马纹、sticky 表头、合计行加重）
- 「更新记录」小节 → 时间线

本目录里三本「示例账本｜…」都是虚构数据，看完渲染效果后可以直接删掉，换成你自己的账本。
