# 更新日志

## v1.1.0 · 2026-07-17

**知识管线改版（本版核心）**

- **阅读器新增「☆ 珍藏」**：珍藏＝标已读＋永久保留；主屏「知识沉淀」卡新增「珍藏」筛选，就是你的珍藏书架
- **已读自动浓缩归档**：点「标记已读」或「珍藏」，后台自动把该篇提炼成要点笔记进 `50-Knowledge/` 对应主题目录；介绍同一个 AI skill 的文章聚合进同一份 `Skill档案/`（一个 skill 永远一份）；浓缩完成后非珍藏原文移入库内 `.trash/`（可恢复，绝不硬删）
- **「AI 浓缩」「AI 清杂」两钮退役**，换单颗「整理」：补漏浓缩、清走已浓缩的非珍藏文章、skill 档案合并去重、散落笔记归位
- **修复**阅读器已读/珍藏按钮第一次点击无反馈、再点反而把状态翻回去的时序 bug（元数据缓存滞后所致，改为本地状态即时响应）

**主屏布局**

- **板块分组**：「日常管理」（待办/日历/项目/账目）与「阅读与知识」（待读文章/知识沉淀）两组玻璃托盘，组内卡片保持独立
- **组级软糖把手**：平时隐身，鼠标滑到托盘底缘才现身；拖动统一调整组内全部卡片高度（按组记忆，双击复位）
- 待读文章一页 15 篇＋糖霜翻页钮；知识沉淀分类筛选（珍藏/设计与配色/Skill/AI与工具/其他）＋翻页
- 追剧墙默认显示「在追」；所有内滚动区（弹层/选季网格/日期面板/卡内列表）滚动条统一糖霜化

**文档**

- 《给智能体的说明书》《接入AI指南》《安装指南》《秘书指南》、AGENTS.md 全部同步新管线；`starred`/`distilled` 字段规则入册（Agent 保留勿动）

## v1.0.8 · 2026-07-16

**界面与手感**

- **行增删丝滑三件套**：新建待办/日程从上方浮入；完成/删除/标记已读先平滑折叠再写数据，下方内容不再瞬跳；重渲染时挪位的行由 FLIP 从旧位置滑到新位置（待办、日历、文章全接线，尊重系统减动效偏好）
- **全屏方章＋禅模式**：顶栏新增全屏按钮（薄荷绿，同步左侧），进入系统级全屏并隐藏 Obsidian 全部外壳只留中枢；绿灯钮/快捷键全屏不受影响，退出自动恢复
- **天气城市不再写死东京**：首次启动按网络 IP 自动定位；点天气卡的城市胶囊（别针小药丸）可搜索换城市（中文/拼音/英文），选一次永久记住
- **主滚动条糖霜化**：糖果渐变细圆条，悬停全饱和；追剧墙横条同族
- **按钮底色护甲**：修复 Obsidian 本体按钮样式在部分主题（如 Baseline 深色）下把糖霜按钮压成黑块的问题
- 问候行防折行、撤掉 ☀︎/☾ 装饰字符；天气三格微调；天气小剧场点击穿透修复

**给 Agent 的文档体系（记账全 AI 化）**

- 新增 **`docs/记账指南.md`**：写给 Agent 的记账手册——schema、呈现契约（写什么形状的表格渲染成占比条/统计胶囊/对账单/时间线）、新账本骨架、常见任务菜谱、红线自查
- 新增 **`docs/秘书指南.md`**：教你的 Agent 当私人秘书的总纲——一句话装机/上岗、六大职责场景表、通用守则
